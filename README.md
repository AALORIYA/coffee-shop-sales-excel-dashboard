# ☕ Coffee Shop Sales Analysis Dashboard

An interactive Coffee Shop Sales Analysis Dashboard created to analyze retail sales data and generate actionable business insights.

This project helps understand customer behavior, sales performance, product trends, store performance, and peak sales timings using data analytics and visualization techniques.

---

# 📌 Project Objective

The main objective of this project is to analyze retail sales data to gain actionable insights that can improve the performance of the coffee shop business.

# 🔍 Key Business Questions Solved

- How do sales vary by day of the week and hour of the day?
- What are the peak sales hours?
- What is the monthly sales revenue?
- Which store location performs best?
- What is the average bill per customer?
- Which products generate the highest revenue?
- How do product categories perform?
- What is the distribution of order sizes?

---

# 📈 Dashboard Insights

## ✅ Sales Performance
- Total Sales: $698,812.33
- Total Footfall: 149,116
- Average Bill Per Person: 4.69
- Average Orders Per Person: 1.44

## ✅ Peak Sales Timing
- Maximum orders are generated during morning hours.
- Sales activity starts increasing sharply from 7 AM.

## ✅ Best Performing Category
- Coffee contributes the highest percentage of total sales.

## ✅ Store Performance
- Hell's Kitchen generated the highest revenue among all store locations.

## ✅ Best Selling Products
Top products based on sales:
- Barista Espresso
- Brewed Chai Tea
- Hot Chocolate
- Gourmet Brewed Coffee
